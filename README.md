# AnnonyBlog-MERN
## A Full stack blog application created with 
- Node
- Express 
- React
- MongoDB
- JSONWebToken
- Bcrypt

## Functionalities 
- Register
- Login
- Create Post
  - Add Title, Summary and Upload Cover Image
- Add Comment
- Like Post
